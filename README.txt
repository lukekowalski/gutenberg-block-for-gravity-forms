Gravity Forms selector for Gutenberg editor
 
== Description ==

A simple plugin that creates Gravity Forms selector in Gutenberg editor.

 
== Installation ==
 
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation.
2. Make sure thaty you have 'Gutenberg' and 'Gravity Forms' plugins installed.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. In Dashboard go to Forms > Settings > Web API and enable Web API
4. Go to Gutenberg editor, play around with the block and have fun.
